# RestAPI_NodeJS
1. Install NodeJS
2. Change to project root directory
3. "npm install"
4. "npm install --save mysql2"
5. "npm install -g nodemon"
6. "nodemon bin/www"
7. Go to http://localhost:3000 on your browser
